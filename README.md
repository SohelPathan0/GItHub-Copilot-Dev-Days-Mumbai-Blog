## 🎯 About the Event

**GitHub Copilot Dev Days | Mumbai** was a hands-on, in-person, community-led event focused on helping developers explore real-world AI-assisted coding using **GitHub Copilot**. As part of the **global GitHub Copilot Dev Days series**, the event brought together **84 developers** from across Mumbai to learn, build, and connect.

## 🎓 Certificate Verification

Every participant received a **digital certificate** with a unique ID and QR code linking to this verification system — powered by **GitHub Pages**.

<div align="center">

### 🌐 Verification Portal

**[sohelpathan0.github.io/certificate-verification](https://sohelpathan0.github.io/certificate-verification/)**

<br/>

### 🔗 Verify a Specific Certificate

```
https://sohelpathan0.github.io/certificate-verification/?id=GCDM-001
```

> Replace `GCDM-001` with the certificate ID printed on your certificate.  
> IDs range from **GCDM-001** through **GCDM-084**

## ⚙️ How It Works

```
┌──────────────────────────────────────────────────────────────────┐
│                                                                  │
│   Certificate (PDF / Image)                                      │
│      ├── Unique URL  →  ?id=GCDM-XXX  ──┐                        │
│      └── QR Code     →  same URL      ──┤                        │
│                                          ↓                       │
│                         Page loads & reads ?id= from URL         │
│                                          ↓                       │
│                         Fetches data.json from this repo         │
│                                          ↓                       │
│                              Looks up certificate ID             │
│                                          ↓                       │
│                         Shows name, event, role, status          |
│                                         OR                       │
│                               Certificate not found              │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

</div>

### 📄 Certificate Record Format (`data.json`)

```json
{
  "certificates": [
    {
      "id": "GCDM-001",
      "name": "Participant Full Name",
      "event": "GitHub Copilot Dev Days | Mumbai",
      "role": "Participant"
    }
  ]
}
```
